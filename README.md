# pos_tagger

A Part Of Speech (POS) Tagger. It relies for now on `Cadmium::Classifier::Viterbi`.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     pos_tagger:
       github: cadmiumcr/pos_tagger
   ```

2. Run `shards install`
